---
title: 'Health Configuration'
openapi: 'GET /health/configuration'
---

Returns current configuration status without exposing sensitive values.

## Example Request

```javascript
const response = await fetch('https://api.skop.dev/health/configuration')
const config = await response.json()
console.log(config)
```

## Response (200 OK)

```json
{
  "status": "healthy",
  "configuration": {
    "database_configured": true,
    "redis_configured": true,
    "external_services_configured": true,
    "environment": "production"
  }
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `status` | string | Configuration status |
| `configuration` | object | Configuration check results |

### Configuration Object

| Field | Type | Description |
|-------|------|-------------|
| `database_configured` | boolean | Database connection configured |
| `redis_configured` | boolean | Redis connection configured |
| `external_services_configured` | boolean | External services configured |
| `environment` | string | Current environment |
