---
title: 'Health Readiness'
openapi: 'GET /health/ready'
---

Kubernetes-style readiness check. Returns `200` if ready, `503` if not ready.

## Example Request

```javascript
const response = await fetch('https://api.skop.dev/health/ready')
const readiness = await response.json()
console.log(readiness)
```

## Response

### Ready (200 OK)

```json
{
  "status": "ready",
  "message": "Service is ready to accept requests"
}
```

### Not Ready (503 Service Unavailable)

```json
{
  "status": "not_ready",
  "message": "Service is not ready to accept requests",
  "reasons": [
    "Database connection not established",
    "External dependencies unavailable"
  ]
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `status` | string | Readiness status (ready, not_ready) |
| `message` | string | Human-readable status message |
| `reasons` | array | List of reasons if not ready (optional) |
