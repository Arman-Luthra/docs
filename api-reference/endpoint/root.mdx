---
title: 'Root Endpoint'
openapi: 'GET /'
---

Returns basic API information and available endpoints.

## Example Request

```javascript
const response = await fetch('https://api.skop.dev/')
const info = await response.json()
console.log(info)
```

## Response (200 OK)

```json
{
  "message": "Skop API",
  "version": "1.0.0",
  "docs": "/docs",
  "health": "/health",
  "endpoints": {
    "scrape": "/scrape",
    "status": "/scrape/status/{job_id}",
    "results": "/scrape/results/{job_id}",
    "cancel": "/scrape/{job_id}",
    "health": "/health"
  }
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `message` | string | Service name identifier |
| `version` | string | Current API version |
| `docs` | string | Documentation endpoint path |
| `health` | string | Health check endpoint path |
| `endpoints` | object | Available API endpoints |
